# Vidly Django App

Mine experiments with Phyton using Code with Mosh course.

## Select virtual env

<leader>cv in NVim.

## Run shell

```sh
pipenv shell
```

## Install dependencies

```sh
pipenv install
```

## Run app or file

```sh
phyton {file_name.py}
```

## Generate Django project

```sh
django-admin startproject vidly .
```

## Generate Django app

```sh
python manage.py startapp movies
```
